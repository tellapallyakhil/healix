## 🌟 Vision
AyuSphere aims to eliminate high-risk self-diagnosis behaviors by providing accurate, clinical-grade cardiac triage at a fraction of the cost. With a viability rating of **91.11%**, we bridge the gap between rural isolation and institutional healthcare.

---

## 🚀 Core Features

### 💎 Premium Digital Deck
A high-fidelity landing page reflecting a "Classic Grey and Beige" visual identity, designed for innovation and entrepreneurship presentations.

### 💓 Live Biometric Logic
Real-time simulation of heart rate biometrics, demonstrating the feasibility of predictive arrhythmia detection through IoT integration.

### 🧠 Scientific Methodology
- **Data Acquisition:** Secure biometric capture via hardware sensors and ThingSpeak MQTT.
- **LSTM Recognition:** Deep learning neural networks optimized for cardiac anomaly identification.
- **RAG Interaction:** Medical-LLM providing vetted, context-aware health guidance.

## 🛠 Getting Started

### Prerequisites
- Node.js 18.x or higher
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/tellapallyakhil/healix.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
